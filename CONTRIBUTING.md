# Contributing to LinkedIn Organic Content Strategy for B2B SaaS

Thank you for your interest in contributing to this research project! 🎉

This is a portfolio research project by **Nihal Dhoke** focused on documenting LinkedIn organic content strategy for B2B SaaS. Contributions that improve the quality, depth, or accuracy of the research are welcome.

---

## 📋 How to Contribute

### 1. Reporting Issues

If you find:
- Incorrect information about an expert or their frameworks
- Broken or outdated URLs
- Missing attribution for a framework or quote
- Factual errors in the algorithm documentation

Please [open an issue](https://github.com/nihaldhoke27-oss/LinkedIn-organic-content-strategy-for-B2B-SaaS/issues) with:
- A clear title describing the issue
- The file and section where the issue appears
- The correct information and a source URL

### 2. Suggesting New Experts

If you'd like to suggest an additional expert for the research:

Open an issue with the tag `expert suggestion` and include:
- Expert's full name
- LinkedIn profile URL
- Website / newsletter
- Why they are relevant to B2B SaaS LinkedIn organic strategy
- 2–3 examples of their publicly available content

### 3. Adding Research

To contribute a new expert profile or research file:

1. Fork the repository
2. Create a new branch: `git checkout -b research/expert-name`
3. Follow the existing file structure (see any file in `/research/linkedin-posts/` as a template)
4. Ensure all sources are cited with URLs and access dates
5. Do not fabricate posts, transcripts, or quotes — clearly note when content is unavailable
6. Submit a pull request with a clear description of what you've added

---

## 📐 Research Standards

All research files in this project must meet the following standards:

### Accuracy
- Only reference publicly available information
- Include source URLs for all factual claims
- Clearly label unavailable content (e.g., transcript not available — see official video link)
- Never fabricate quotes, posts, or data

### Attribution
- Credit every framework, quote, and methodology to its creator
- Include a Sources & Attribution table at the end of every file
- Note the date the source was accessed

### Format
- Use the established markdown structure (see existing files as reference)
- Include all standard sections: Bio, Why Selected, Posting Frequency, Content Themes, Writing Patterns, Post Summaries, Key Takeaways, Hooks Library, Sources
- Keep sentences clear and actionable — this is a professional research document

### Scope
- Content must be relevant to **LinkedIn organic growth** and **B2B SaaS marketing**
- Personal or unrelated social media strategy is out of scope
- All experts should have publicly demonstrated results in the B2B SaaS space

---

## 🤝 Code of Conduct

- Be respectful in all interactions
- Credit original creators for their work
- Do not submit AI-generated content presented as research without human review and attribution
- Do not include private or non-public information about any individual
