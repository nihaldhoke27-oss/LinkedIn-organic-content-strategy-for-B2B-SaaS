# LinkedIn Organic Content Strategy for B2B SaaS

Author: Nihal Dhoke

## Objective

This repository documents research on LinkedIn Organic Content Strategy for B2B SaaS.

The project will include:

- 10 industry experts
- LinkedIn post research
- YouTube transcript research
- Content pattern analysis
- Best practices
- Actionable insights

## Repository Structure

```
/
├── README.md                          # Project overview, objectives, and status
└── research/
    ├── sources.md                     # Expert profiles, articles, and reference links
    ├── linkedin-posts/                # Collected LinkedIn posts from industry experts
    │   └── README.md
    ├── youtube-transcripts/           # YouTube video transcripts for content strategy research
    │   └── README.md
    └── other/                         # Additional research materials and notes
        └── README.md
```

### Folder Descriptions

| Folder / File | Purpose |
|---------------|---------|
| `README.md` | Main project documentation with objectives, structure, and current status. |
| `research/sources.md` | Central index of industry experts, articles, and reference materials. |
